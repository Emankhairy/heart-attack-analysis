### Title
## Heart Attack Data Analysis

### Description
This project contains a Jupyter Notebook (heart_attack.ipynb) that analyzes a heart attack dataset to understand patterns related to age, gender, cholesterol, blood pressure, and other medical factors. The analysis uses Python, Pandas, Matplotlib, and Seaborn to generate insights and visualizations.

### Contents
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Age distribution
- Gender ratio
- Cholesterol & blood pressure exploration
- Visualizations (bar plots, histograms, etc.)

### Requirements
pip install pandas numpy matplotlib seaborn
